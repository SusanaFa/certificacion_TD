## Instalación de dependencias varias

1. `npm init -y`
2. `npm i nodemon`
3. `npm i express`
4. `npm i body-parser`
5. `npm i dotenv`
6. `npm i pg`
7. `npm i sequelize`
8. `npm i jsonwebtoken`
9. `npm i bcrypt`
